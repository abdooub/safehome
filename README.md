# SafeHome
Le projet SafeHome est une application web simple qui permet à l’utilisateur de surveiller sa maison et de contrôler certains équipements à distance. 

L’objectif est de proposer une interface claire, facile à utiliser, et adaptée au format Desktop. 

Le site doit offrir des fonctionnalités de base comme : 

la surveillance (caméra simulée), 
la gestion des alertes, 
le contrôle des appareils, 
et un historique des actions effectuées. 
 

Ce projet sera développé par une équipe de 6 personnes dans un cadre pédagogique.
